# 🕹️ MEGAMANIA REMAKE — Atari Style

Remake do clássico jogo de tiro espacial **Megamania** do Atari 2600, desenvolvido em HTML5 Canvas puro com estética **pixel art 8-bit**.

## 🎮 Como jogar

| Tecla | Ação |
|-------|------|
| `←` / `→` | Mover a nave |
| `Espaço` | Atirar |

## ✨ Funcionalidades

- **5 tipos de inimigos** desenhados em pixel art: hambúrgueres, biscoitos, ferros de passar, gravatas borboleta e diamantes
- **Barra de energia** que drena constantemente — destruir uma onda a reabastece
- **3 vidas** com animação de explosão ao morrer
- **Progressão de níveis** com inimigos cada vez mais rápidos
- **Efeitos sonoros** gerados via Web Audio API (sem arquivos externos)
- **Fundo com estrelas** em parallax
- **Score e Hi-Score** persistidos na sessão

## 🚀 Como executar

Basta abrir o arquivo `index.html` diretamente no navegador — sem instalação, sem servidor, sem dependências!

## 🛠️ Tecnologias

- HTML5 / Canvas 2D
- JavaScript puro (Vanilla JS)
- Web Audio API
- CSS com fonte Google Fonts (Press Start 2P)

## 📸 Inspiração

Baseado no jogo **Megamania** (1982) da Activision para Atari 2600.
